# Studio Competitions: Gamifying Entire Portfolios of Autonomous Startups

Managing a portfolio of autonomous startups in 2026 presents a unique challenge: How do you create cohesion, healthy competition, and knowledge sharing across independent founders? Studio competitions provide the answer.

## The Studio Portfolio Challenge

### Traditional Portfolio Management

Studios typically manage portfolios through:
- Quarterly board meetings
- Monthly check-ins
- Slack channels (low engagement)
- Occasional events
- Shared resources

Result: Founders operate in silos, minimal cross-pollination, missed opportunities for synergy.

### The Autonomous Business Multiplier

In 2026, with autonomous businesses:
- Less operational oversight needed
- More founders per partner possible
- Portfolio sizes growing (20-50+ companies)
- Geographic distribution increasing
- Interaction naturally decreasing

How do you maintain studio culture and drive performance at scale?

```typescript
export const getStudioLeaderboard = codeFunction({
  input: z.object({
    studioId: z.string(),
    leaderboardType: z.string()
  }),
  async execute({ studioId, leaderboardType }) {
    const founders = await getStudioFounders(studioId);
    const config = studioLeaderboards[leaderboardType];

    const rankings = await Promise.all(
      founders.map(async founder => ({
        founderId: founder.id,
        name: founder.name,
        score: await calculateMetric(founder.id, config.metric),
        trend: await getTrend(founder.id, config.metric, 7),
        badge: founder.badges
      }))
    );

    rankings.sort((a, b) => b.score - a.score);

    return {
      leaderboard: rankings,
      updated: Date.now(),
      nextUpdate: calculateNextUpdate(config.updateFrequency)
    };
  }
});
```

## Team Dynamics

### Cross-Portfolio Teams

Create teams across different startups:

```typescript
export const createCrossPortfolioTeam = codeFunction({
  input: z.object({
    studioId: z.string(),
    teamName: z.string(),
    founderIds: z.array(z.string()),
    purpose: z.string()
  }),
  async execute({ studioId, teamName, founderIds, purpose }) {
    // Verify all founders are in studio
    const founders = await getFounders(founderIds);
    const allInStudio = founders.every(f => f.studioId === studioId);

    if (!allInStudio) {
      throw new Error('All team members must be in same studio');
    }

    const team = await createTeam({
      studioId,
      name: teamName,
      members: founderIds,
      purpose,
      type: 'cross-portfolio',
      features: {
        sharedResources: true,
        knowledgeExchange: true,
        jointMetrics: true,
        collaborativeGoals: true
      }
    });

    // Enable team-specific features
    await enableTeamFeatures(team.id);

    // Create team leaderboard
    await createTeamLeaderboard(team.id);

    return team;
  }
});
```

```typescript
export const trackStudioChallenge = agenticFunction({
  schedule: 'daily',
  async execute() {
    const challenges = await getActiveStudioChallenges();

    for (const challenge of challenges) {
      const currentValue = await calculateStudioMetric(
        challenge.studioId,
        challenge.goal.metric
      );

      // Update progress
      await updateChallengeProgress(challenge.id, currentValue);

      // Check milestones
      for (const [threshold, milestone] of Object.entries(challenge.milestones)) {
        if (currentValue >= Number(threshold) && !milestone.achieved) {
          await achieveMilestone(challenge.id, threshold);
          await rewardStudio(challenge.studioId, milestone.reward);
          await unlockForStudio(challenge.studioId, milestone.unlocks);
        }
      }

      // Check goal completion
      if (currentValue >= challenge.goal.target) {
        await completeChallenge(challenge.id);
        await celebrateStudioWin(challenge.studioId, challenge);
      }
    }
  }
});
```

```typescript
export const measureCompetitionROI = agenticFunction({
  schedule: 'monthly',
  async execute() {
    const studios = await getAllStudios();

    for (const studio of studios) {
      const founders = await getStudioFounders(studio.id);

      // Compare competition vs non-competition periods
      const metrics = {
        duringCompetitions: await getMetrics(founders, 'competition-periods'),
        outsideCompetitions: await getMetrics(founders, 'normal-periods')
      };

      const impact = {
        executionIncrease: (metrics.duringCompetitions.execution - metrics.outsideCompetitions.execution) / metrics.outsideCompetitions.execution,
        revenueIncrease: (metrics.duringCompetitions.revenue - metrics.outsideCompetitions.revenue) / metrics.outsideCompetitions.revenue,
        activityIncrease: (metrics.duringCompetitions.activity - metrics.outsideCompetitions.activity) / metrics.outsideCompetitions.activity,
        participationRate: metrics.duringCompetitions.participation,
        satisfaction: await getSatisfactionScore(studio.id)
      };

      await saveCompetitionImpact(studio.id, impact);
    }
  }
});
```
