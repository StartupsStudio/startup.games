# Competitions That Matter: How Time-Bound Challenges Push Founders Further

There's something about a ticking clock and visible competition that unlocks superhuman productivity. In 2026, startup.games leverages this with carefully designed competitions that push founders to achieve more in days than they typically do in months.

## The Science of Competition

### Deadline Magic

Parkinson's Law states: "Work expands to fill the time available." Give yourself a month to build a feature, it takes a month. Give yourself a weekend, it's done by Sunday night.

Time-bound competitions exploit this by creating artificial urgency that drives focus:

```typescript
export interface FounderCompetition {
  id: string;
  title: string;
  description: string;
  startDate: string;
  endDate: string;

  objective: {
    metric: string; // 'revenue', 'users', 'features_shipped', etc.
    target?: number;
    relative: boolean; // Compete against others vs. absolute target
  };

  eligibility: {
    minLevel?: number;
    maxLevel?: number;
    cohort?: string;
    inviteOnly?: boolean;
  };

  prizes: {
    first: CompetitionPrize;

    third?: CompetitionPrize;
    participation?: CompetitionPrize;
  };

  rules: string[];
  judging: 'automated' | 'human' | 'hybrid';
}
```

### Competitive Arousal Theory

Competition triggers physiological responses:
- Increased adrenaline
- Heightened focus
- Enhanced performance
- Greater risk tolerance

But too much competition causes anxiety and degraded performance. The key is designing competitions with the right intensity.

## Competition Mechanics

### Automated Leaderboards

Real-time ranking creates constant motivation:

```typescript
export const updateCompetitionLeaderboard = agenticFunction({
  schedule: 'every 10 minutes',
  async execute() {
    const activeCompetitions = await getActiveCompetitions();

    for (const competition of activeCompetitions) {
      const participants = await getCompetitionParticipants(competition.id);
      const rankings = [];

      for (const participant of participants) {
        const score = await calculateCompetitionScore({
          participantId: participant.founderId,
          competitionId: competition.id,
          metric: competition.objective.metric,
          startDate: competition.startDate
        });

        rankings.push({
          founderId: participant.founderId,
          score,
          rank: 0 // Will be calculated after sort
        });
      }

      // Sort by score descending
      rankings.sort((a, b) => b.score - a.score);

      // Assign ranks (handle ties)
      for (let i = 0; i < rankings.length; i++) {
        rankings[i].rank = i + 1;
      }

      await saveCompetitionLeaderboard(competition.id, rankings);

      // Notify participants of rank changes
      await notifyRankChanges(competition.id, rankings);
    }
  }
});
```

### Progress Visibility

Participants can see their standing and progress:

```typescript
export const getCompetitionStatus = codeFunction({
  input: z.object({
    competitionId: z.string(),
    founderId: z.string()
  }),
  async execute({ competitionId, founderId }) {
    const competition = await getCompetition(competitionId);
    const leaderboard = await getCompetitionLeaderboard(competitionId);
    const myEntry = leaderboard.find(e => e.founderId === founderId);

    const timeRemaining = new Date(competition.endDate).getTime() - Date.now();
    const percentComplete =
      (Date.now() - new Date(competition.startDate).getTime()) /
      (new Date(competition.endDate).getTime() - new Date(competition.startDate).getTime());

    return {
      myRank: myEntry?.rank || 'Not ranked',
      myScore: myEntry?.score || 0,
      totalParticipants: leaderboard.length,
      timeRemaining,
      percentComplete,
      topThree: leaderboard.slice(0, 3),
      nearbyCompetitors: leaderboard
        .filter(e =>
          Math.abs(e.rank - (myEntry?.rank || 999)) <= 2
        )
    };
  }
});
```

### Last-Minute Push Notifications

As competitions near end, send strategic reminders:

```typescript
export const sendCompetitionReminders = agenticFunction({
  schedule: 'hourly',
  async execute() {
    const activeCompetitions = await getActiveCompetitions();

    for (const competition of activeCompetitions) {
      const timeRemaining =
        new Date(competition.endDate).getTime() - Date.now();

      const hoursRemaining = timeRemaining / (1000 * 60 * 60);

      // Send reminders at key moments
      if (hoursRemaining <= 24 && hoursRemaining > 23) {
        await notifyParticipants(competition.id, '24-hour-warning');
      }

      if (hoursRemaining <= 6 && hoursRemaining > 5) {
        await notifyParticipants(competition.id, '6-hour-warning');
      }

      if (hoursRemaining <= 1 && hoursRemaining > 0.9) {
        await notifyParticipants(competition.id, 'final-hour');
      }
    }
  }
});
```

## Competition Frequency and Fatigue

Too many competitions cause fatigue:

**Optimal Calendar**:
- Sprint competitions: 1-2 per month
- Weekly challenges: 2-3 per month (staggered)
- Monthly championships: 1 per month
- Quarterly major events: 1 per quarter

```typescript
export const validateCompetitionSchedule = agenticFunction({
  input: z.object({
    proposedCompetition: z.object({
      startDate: z.string(),
      endDate: z.string(),
      intensity: z.enum(['low', 'medium', 'high'])
    })
  }),
  async execute({ proposedCompetition }) {
    // Check for competition overlap
    const existingCompetitions = await getCompetitionsInRange(
      proposedCompetition.startDate,
      proposedCompetition.endDate
    );

    // Don't allow more than 2 high-intensity competitions overlapping
    const highIntensity = existingCompetitions.filter(
      c => c.intensity === 'high'
    );

    if (highIntensity.length >= 2 && proposedCompetition.intensity === 'high') {
      return {
        approved: false,
        reason: 'Too many high-intensity competitions in this period',
        suggestion: 'Consider moving to different dates or lowering intensity'
      };
    }

    return { approved: true };
  }
});
```
