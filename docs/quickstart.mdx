# Quickstart

Get started with Startup Games in under 5 minutes.

## Installation

```bash
npm install startup.games
```

## Connect Your Startup

```typescript
import { Games } from 'startup.games'

// Connect your startup
const player = await Games.connect({
  startup: 'MyStartup',
  player: 'founder@mystartup.com'
})

// Check your stats
console.log(player.level)         // 1
console.log(player.xp)            // 0
console.log(player.achievements)  // []
```

## Start a Quest

```typescript
// View available quests
const quests = await player.quests.available()

// Start the launch quest
await player.quests.start('launch-week')

// Complete quest steps
await player.quests.completeStep('launch-week', 'prepare-launch')
await player.quests.completeStep('launch-week', 'announce')
await player.quests.completeStep('launch-week', 'engage')

// Quest complete!
// ğŸ¯ +500 XP
// ğŸ† Badge: Launcher
```

## Check Your Rank

```typescript
// View your leaderboard position
const rank = await player.leaderboard.rank('xp')
console.log(rank)  // #156 of 1,234 players
```

## Join a Competition

```typescript
// View active competitions
const competitions = await player.competitions.available()

// Join one
await player.competitions.join('launch-challenge-2024')
```
