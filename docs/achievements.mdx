# Achievements

Achievements are badges you unlock for hitting milestones on your startup journey.

## Overview

Every achievement:

- Has a clear unlock condition
- Awards XP when unlocked
- Displays on your profile
- Can be shared with the community

## Viewing Achievements

```typescript
import { Achievements } from 'startup.games'

// View all available achievements
const all = await Achievements.list()

// View by category
const launch = await Achievements.list({ category: 'launch' })
const growth = await Achievements.list({ category: 'growth' })
const revenue = await Achievements.list({ category: 'revenue' })
```

## Automatic Unlocking

With integrations, achievements unlock automatically:

```typescript

import { Games } from 'startup.games'

const startup = await Startup.create({
  name: 'MyStartup',
  gamification: true  // Enable auto-tracking
})
```

## Manual Unlocking

For custom milestones:

```typescript
// Log an event that triggers an achievement
await player.events.log({
  type: 'custom-milestone',
  data: {
    milestone: 'launched-mobile-app',
    platform: 'ios'
  }
})
```
