# Quests

Quests are structured multi-step challenges that guide you through important milestones and reward completion with XP and badges.

## Overview

```typescript
import { Quests } from 'startup.games'

// View all available quests
const all = await Quests.list()

// View quests by category
const launch = await Quests.list({ category: 'launch' })
const growth = await Quests.list({ category: 'growth' })
```

## Your Quests

```typescript
// View available quests (not started)
const available = await player.quests.available()

// View active quests (in progress)
const active = await player.quests.active()

// View completed quests
const completed = await player.quests.completed()
```

## Starting a Quest

```typescript
// View quest details
const quest = await Quests.get('launch-week')

// {
//   id: 'launch-week',
//   name: 'Launch Week',
//   description: 'Execute a successful product launch',
//   category: 'launch',
//   difficulty: 'medium',
//   estimatedTime: '7 days',
//   steps: [
//     { id: 'prepare-launch', name: 'Prepare Launch Assets' },
//     { id: 'announce', name: 'Announce on Social Media' },
//     { id: 'producthunt', name: 'Launch on Product Hunt' },

//   ],
//   rewards: {

//     badge: 'launcher',
//     achievement: 'successful-launch'

// Start the quest
await player.quests.start('launch-week')
```

## Completing Steps

```typescript
// Complete individual steps
await player.quests.completeStep('launch-week', 'prepare-launch')
await player.quests.completeStep('launch-week', 'announce')

// Check progress
const progress = await player.quests.progress('launch-week')

// {
//   id: 'launch-week',
//   status: 'in-progress',
//   steps: [
//     { id: 'prepare-launch', completed: true },

//     { id: 'report', completed: false }

//   progress: 40,  // 2 of 5 steps

```

## Quest Chains

Some quests unlock after completing others:

```typescript
// Complete prerequisite
await player.quests.complete('first-launch')

// New quest unlocked
const newlyAvailable = await player.quests.available()
// Includes: 'scale-launch', 'multi-product-launch', etc.
```

## Abandoning Quests

```typescript
// If you need to abandon a quest
await player.quests.abandon('launch-week')

// Note: Progress is saved if you restart later
```
